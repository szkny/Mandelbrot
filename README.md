# Mandelbrot

Program of Mandelbrot Set using GLUT

Extract zip file, and make
```
$ cd ~/Downloads
$ unzip Mandelbrot-master.zip
$ cd ./Mandelbrot-master
$ make
$ ./bin/Mandelbrot
```
